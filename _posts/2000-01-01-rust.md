Rust to szybki język programowania, zapobiegający naruszeniom pamięci i gwarantujący bezpieczeństwo wątków.

- [The Rust Programming Language] - obowiązkowa lektura wprowadzająca do języka, wyjaśniająca składnię oraz ważne pojęcia.
- [Standard Library API Reference] - dokumentacja biblioteki standardowej.
- [Rust by Example] - zbiór przykładowych programów ilustrujących różne koncepcje Rusta.
- [The Rustonomicon] - książka dla bardziej zaawansowanych, wyjaśniająca jak pisać kod z użyciem `unsafe`.
- [rust-learning] - szeroki zbiór dostępnych materiałów do nauki Rusta.
- [Discovery] - wprowadzenie do programowania mikrokontrolerów (na przykładzie zestawu STM32F3DISCOVERY) z użyciem Rusta.

[Standard Library API Reference]: https://doc.rust-lang.org/std/
[Rust by Example]: https://doc.rust-lang.org/rust-by-example/
[The Rustonomicon]: https://doc.rust-lang.org/nomicon/
[rust-learning]: https://github.com/ctjhoa/rust-learning
[Discovery]: https://rust-embedded.github.io/discovery/
